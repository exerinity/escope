What is escope?

escope (a backronym for Ephemeral Scoped Control of Persistent Endpoints) is a minimalist URL shortener/redirect service where you create ephemeral redirect links, or "scopes". Think of it as Bit.ly, but the links always die automatically after a while. That while can be as short as 1 minute and as long as 3 days, or 4320 minutes.

Why?

escope can have many use cases. Honorable mentions:

  • Downloading extremely specific builds of software on VMs where the VM can't read the host clipboard
  • Showing people extremely long links, especially fraudulent ones in scambaiting
  • In tandem use with disappearing messages on apps like Telegram

Okay, how does it work?

escope is written in TypeScript and runs on Cloudflare Workers. It uses KV for scope storage. The codebase is very lean and thus is the best of both worlds: the edge-computing speed of Workers and a very lightweight source tree (or whatever)

Can I use the API in my projects?

Yes. Responsibly. If you spam links, the rate limiter will nuke you pretty quickly. If you use escope to make a botnet of unknown links, you will also be blocked across all IP addresses.
